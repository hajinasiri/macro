    • id

      Camelcase alphanumeric, no special characters or punctuation marks except for ‘-’ dash.

      Usually just a a version of the label with spaces and special characters removed - only dashes allowed.

    • label

      Text label for the UNO: used in the info-pane and tooltips.

      This is the text from Scrivener’s ‘Title’.

    • subtitle

      An optional subtitle that will be appended to the label in the map. Having a separate subtitle helps to keep the IDs shorter, and allows for formatting the subtitles differently in the map.

    • outlineNumber

      Period separated outline number e.g. 4.2.3.4

    • outlineLevel

      Indent level of the outline e.g. 4 for the outlineNumber 4.2.3.4

    • parent

      The id of the parent, e.g. the Title / label stripped of illegal characters

    • classes

      Classes are assigned in Scrivener using Keywords.

      Assigns CSS classes to the UNO. Can be any class name used by the designer, or the special classes:

      - node: indicates that this is a ‘thing’ UNO. It may also be a .view link in the story pane.

      - link: indicates this is connection between things

      - view: this will be a ‘slide’ link that is included in the story-pane. It make also be a .node with artwork in the map.

      - voa: marks this as a voice-over animation link

      - connection: this uno will in...

    • unoFrom

      Used by the sequencing function to identify the from UNO.

    • unoTo

      Used by the sequencing function to identify the to UNO.

    • param1

      A map specific variable to be used as needed.

    • param2

      A map specific variable to be used as needed.

    • param3

      A map specific variable to be used as needed.

    • param4

      A map specific variable to be used as needed.

    • shortDescription

      Markdown text for the tooltip.

      This is the text from Scrivener’s ‘Synopsis’.

    • longDescription

      Markdown text for the info-pane.

      This is the text from Scrivener’s main ‘doucument’ window (what get’s saved as the separate .rtf files).

    • hoverAction

      both (default)

      open

      close

      function

    • hoverFunction

      functionName=params

      A function to be run on hovering - in URL command syntax.

    • clickAction

    • clickFunction

      functionName=params

      A function to be run on clicking - in URL command syntax.

    • onDoubleClick

      functionName=params

      A URL command to run when double-clicking on an UNO.

    • url

      A URL used by a custom function and/or by the context menu, in controlmenu.js, as a link in the right-click menu.

    • urlText

      A text label for the url link: used by a custom function and/or by the context menu, in controlmenu.js, as a link in the right-click menu.

    • tooltip

      0 | 1

      Do not show (0) or show (1) the tooltip.

    • infoPane

      none

      hover

      click

    • onURL

      An optional URL command that is run when UNO is turned on.

    • offURL

      An optional  URL command that is run when UNO is turned off.

    • openURL

      An optional  URL command that is run when UNO opened.

    • closeURL

      An optional  URL command that is run when UNO closed.

    • onFunction

      An optional function that is run when the UNO is turned on – replaces the default function: display: inline.

    • offFunction

      An optional function that is run when the UNO is turned off – replaces the default function: display: none.

    • openFunction

      An optional function that is run on this UNO’s xxx or vvv when it’s opened – replaces the default function: display: inline.

    • closeFunction

      An optional function that is run on this UNO’s xxx or vvv when it’s closed – replaces the default function: display: none.

    • ttStyle

      The name of the qtip2 tooltip style specification in customstuff.js

    • slideURL

      This will be the URL triggered for ‘slide’ (or ‘view’) links in the story pane.

    • render

      render = ""X"" if UNO layer exists delete it and replace it with a new one

      render = ""N"" if UNO layer exists skip the UNO entirely and only render new UNOs

    • symbol

      If oookind is 'symbol' then this is specifies an Illustrator symbol name - else - for 'rect' or 'ellipes' it specifies a Graphic Style. If blank nothing will be put in the ooo group.

    • location

      Where to place the UNO within its parent:

      - uno

      - xxx

      - vvv

    • xpos

      Horizontal position from the upper left corner of the artboard to the center point of the symbol. If the Kind is Radial - this value is the radius

    • ypos

      Vertical position from the upper left corner of the artboard to the center point of the symbol - in positive points (going down) from the upper left corner. If the Kind is Radial - this value is the offset angle from 0 degrees.

    • xsize

      In points

    • ysize

      In points

    • xoffset

      Horizontal distance between siblings - in points.

    • yoffset

      Vertical distance between siblings - in points.
