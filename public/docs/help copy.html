User Guide

Map Objects

Map objects are hierarchically organized can be opened/closed to show/hide their 'child-objects'.

hover or click on an object to open/close that object.
  on an object to open/close all levels.
  on an object to open and zoom-in on that object.
When you hover or click on certain map objects, additional information about that object will appear in a tooltip and/or here in the 'info-pane'.

Navigating

[ ]   bracket keys to zoom out / in
\ backslash key to zoom-to-fit
arrow keys to pan

Changing 'Perspectives'

Perspective provide a set of pre-defined view of the map.

The comma and period keys:
< >
Will move forward / backward through the different Perspectives. Shift + comma/period for first/last Perspective.

Panes

You can size this pane, and the 'story-pane' to the left, by dragging the dividers. The panes can also be toggled open/close by clicking-on the dividers or by the

e   key - toggles the 'east' pane (the info-pane) open/closed
w  key - toggles the 'west' pane (the story-pane) open/closed
